# EXPERIMENT--06-IoT-Based-Relay-Control-System-Using-LoRaWAN-and-Application-Server
## Aim
To configure a LoRaWAN end device and visualize relay status using an application server dashboard.

## Components Required
- LoRaWAN end device
- Relay module
- Gateway
- Application server dashboard

## Procedure
1. Configure the LoRaWAN device.
2. Connect the relay module to the device.
3. Send relay status data to the application server.
4. Create a dashboard with a light bulb icon.
5. Display ON/OFF status based on relay control.

## Output
The dashboard displays the relay state using a bulb icon:
- Bulb ON → Relay ON
- Bulb OFF → Relay OFF

## Conclusion
The experiment demonstrates remote relay monitoring and control using LoRaWAN and an application dashboard.
